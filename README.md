PERSONAL PASSWORD MANAGER


How-To Setup
1. Python is installed on device https://www.python.org/downloads/
2. Open command window
3. Use the following command to install the python cryptography library
```
pip install cryptography
```
4. Download password_manager.py to a folder of your choosing
5. Run the program, this will create both a 'secret.key' and 'passwords.db' file
6. When not using the program, store the secret.key file in a seperate folder for increased security 
  
That is all!
